# Disaster Robotics Framework

An engineering framework for resilient human-supervised disaster robotics operating under communication failures.

## Overview

This repository accompanies the research paper and supporting materials developed by Geet Lagad, an ISC CLass XII Science student at St. Mary's Junior College, Pune, India.

The framework explores resilient architectures for autonomous and semi-autonomous robotic systems deployed in disaster response, with emphasis on 
- degraded communications environments
- human oversight and decision support
- scalable multi-agent coordination
- resilient mission execution
